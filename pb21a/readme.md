# X.Tips Palmbox 21a Keyboard

* Keyboard Maintainer: X.Tips
* URL: www.umux.com

> PB21a.2040 MCU: rp2040

2 modes go to bootloader:
1. The Bootmagic is enable, To go to bootloader press and hold the left key in the top row of the keyboard, while plugging in. If you edit the fireware, please keep the bootmagic feature or set a QK_BOOTLOADER keycode in your keymap file. It can put the keyboard into bootloader mode for flashing.
2. Open the back cover and quickly short RUN twice to enter bootloader mode.

> PB21a.072c MCU: stm32f072c8t6

2 modes go to bootloader:
1. The Bootmagic is enable, To go to bootloader press and hold the middle key in the top row of the keyboard, while plugging in. If you edit the fireware, please keep the bootmagic feature or set a QK_BOOTLOADER keycode in your keymap file. It can put the keyboard into bootloader mode for flashing.
2. Open the back cover and short BOOT pads on PCB while plugging in PC.
