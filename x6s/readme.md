# X.Tips X6s Split Keyboard

* Keyboard Maintainer: X.Tips
* URL: www.umux.com

---

1. The **Bootmagic** is enable, To go to bootloader press 3 or 8 while plugging in PC.
2. If you edit the fireware, please keep the **Bootmagic** feature or set a **QK_BOOTLOADER** keycode in your keymap file. It can put the keyboard into bootloader mode for flashing.
3. 072c/401c: Short the BOOT pads on the PCB while plugging in PC. 103c: After connecting the keyboard to the computer, quickly short the reset pads on the PCB twice.
